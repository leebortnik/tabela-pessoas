# CRUD SD

O CRUD SD é um sistema web de cadastro de pessoas desenvolvido com HTML, CSS e JavaScript, utilizando armazenamento local do navegador para gerenciamento de dados. O projeto permite cadastrar, editar e excluir usuários de forma simples e intuitiva, apresentando as informações em uma tabela organizada e responsiva.

O sistema conta com validações inteligentes, como verificação de idade permitida e formatação automática de CPF, melhorando a experiência do usuário durante o preenchimento dos dados.

Além da funcionalidade CRUD (Create, Read, Update e Delete), o projeto também possui uma interface moderna e responsiva, oferecendo melhor visualização em diferentes dispositivos.

Tecnologias utilizadas:

* HTML
* CSS
* JavaScript

Funcionalidades:

* Cadastro de pessoas
* Edição de registros
* Exclusão de registros
* Validação de CPF
* Máscara automática de CPF
* Validação de idade
* Interface responsiva
* Armazenamento local dos dados
